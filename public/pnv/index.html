Phone Number Verification

About

Phone number verification for Android using the Open Gateway operator network.

Phone Number Verification is an Android app by Cuatro Studio that demonstrates phone number verification via the Open Gateway operator token standard. It uses Firebase Phone Number Verification and the Android Credential Manager to verify a device's phone number directly through the mobile carrier — without SMS.

How it works

  • Enter your phone number in the app
  • The app requests verification via the Android Credential Manager
  • The carrier authenticates the device silently over the mobile network
  • A verified phone number token is returned without any SMS or user friction
