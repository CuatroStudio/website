Better Clipboard

Latest release

Clipboard history, pins, previews

Better Clipboard is a fast clipboard app and manager for macOS that saves your copy and paste history in a floating window. Instantly browse, search, and pin clipboard items using a keyboard shortcut or menu bar icon. Built for speed, focus, and everyday productivity on macOS.

You choose what to copy. Better Clipboard helps you keep it, reuse it, and make it better when you need to. On supported Macs, Better Clipboard can trigger Apple Intelligence system features such as rewriting and translation. All processing is handled by macOS. No cloud processing. Translation is available on macOS 15+ where supported.

Better Clipboard is a lightweight clipboard app designed for daily macOS workflows. Unlike basic clipboard tools, Better Clipboard shows your history as animated cards instead of plain lists, making it faster to scan and easier to use. Instantly paste text, emojis, images, and programming code you copied earlier without re-copying or switching apps.

Features

  • Quickly revisit recent copies or search your full copy history
  • Paste any previous copy, not just the most recent one
  • Copy and paste text, links, images, emojis, and code
  • Link previews using website meta tags
  • Inline images and readable code snippets
  • Emojis are shown as large, easy-to-recognize images
  • Keyboard shortcuts for faster paste
  • Menu bar access with lightweight performance

Free vs Pro

The free version stores up to 10 clipboard entries and allows up to 3 pinned items. Unlock Better Clipboard Pro for unlimited history and unlimited pins.
